# 🎬 CineScope

CineScope is a stylish and responsive movie browsing app built with **React**, **React Router**, and **Tailwind CSS**. It uses local JSON data and movie posters to simulate an offline movie exploration experience — no API required!

![CineScope Banner](public/posters/hero.jpg)

---

## 🚀 Features

- 🧭 Dynamic routing with React Router
- 🎞️ Detailed movie pages with genre badges and descriptions
- 💅 Fully responsive design using Tailwind CSS
- 🎲 Random movie carousel on Home page
- 📦 Local movie data (`movies.json`) and local poster images
- 📄 Clean and modern UI/UX

---

## 🛠️ Built With

- [React](https://reactjs.org/)
- [React Router DOM](https://reactrouter.com/)
- [Tailwind CSS](https://tailwindcss.com/)
- Local JSON data (no API)
- Static image assets
